## 🛠️ Deploying to GitHub Pages

1. **Create a new repository** on GitHub (e.g., `sierra-oreilly-microsite`)
2. **Upload both files** (`index.html` and `README.md`) to the root of the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Select branch: `main` — folder: `/ (root)`
6. Click **Save**
7. GitHub will provide a live URL within 1–2 minutes

> No build step, no framework, no dependencies required. The site is fully self-contained and loads all fonts from Google Fonts via CDN.
